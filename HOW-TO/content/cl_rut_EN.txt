<p>Every identifier produced by the cl_rut function adheres strictly to official Chilean standards. The underlying algorithm generates a sequence of seven to eight digits followed by a verification digit, which is calculated using the Modulo 11 (MOD-11) check system. This check digit can be a numerical value from zero to nine or the uppercase letter "K." To ensure the test data looks authentic and matches real-world usage, mock-jutsu automatically formats the output with the characteristic periods and hyphen, such as 12.345.678-9. This precision makes the generated strings immediately compatible with front-end input masks and backend database validation constraints.</p>
